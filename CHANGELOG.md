# Changelog

All notable changes to the public portfolio are recorded here.

## 1.1.0 — 2026-07-18

### Added

- Real desktop, mobile and current-teaching product captures generated from the application.
- A 1280 × 640 social-preview PNG composed from the real interface.
- A compact three-scene SVG product tour with a static reduced-motion fallback.

### Changed

- Replaced every placeholder-domain canonical, Open Graph, Twitter and JSON-LD URL with the verified GitHub Pages URL.
- Added complete social-image dimensions, type and alternative text.
- Added explicit lint, type-check, metadata-test and build gates to deployment CI.
